# Chrome Web Store Asset Checklist

## Required

- `128x128` extension icon
- At least one Chrome Web Store screenshot

## Recommended

- Screenshot of the popup home state
- Screenshot of the active capture state
- Screenshot of the session library
- Screenshot of the session detail or review page
- Screenshot of the downloaded folder structure in Chrome Downloads
- Optional promotional image if desired

## Screenshot Guidance

- Avoid showing private or sensitive browsing data.
- Use a demo webpage or a public webpage.
- Make the value obvious from the screenshot.
- Show the “Frames saved to Chrome Downloads” messaging where possible.
- Prefer clean example sessions and readable filenames.
