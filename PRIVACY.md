# ScrollSnap Privacy Policy

ScrollSnap is designed to work locally on the user’s device.

## What ScrollSnap Does

- ScrollSnap captures screenshots only when the user clicks `Start Capture`.
- Captured frames are saved locally through Chrome Downloads.
- The extension may store lightweight session metadata and optional preview thumbnails locally in the browser so the session library and review screens work.

## What ScrollSnap Does Not Do

- Screenshots are not uploaded.
- No backend server is used.
- No cloud sync is used.
- No analytics are used.
- No tracking is used.
- No user data is sold.
- No data is shared with third parties.

## Local Storage and Downloads

- ScrollSnap stores downloaded screenshots on the user’s computer through Chrome Downloads.
- ScrollSnap may also keep lightweight local metadata, settings, and optional preview thumbnails inside browser storage.
- Deleting a session from the ScrollSnap library does not delete files already downloaded to the user’s computer.
- Users can delete downloaded files manually from their computer.
- Users can remove extension-local metadata through the extension UI.

## Limited Use Statement

ScrollSnap’s use of user data is limited to providing the user-facing screenshot capture, local session history, and export functionality described in the extension. ScrollSnap does not use or transfer user data for advertising, analytics, creditworthiness, or unrelated purposes.
