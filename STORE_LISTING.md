# ScrollSnap

## Short Description

Capture clean, ordered screenshot frames while you scroll webpages and browser documents.

## Detailed Description

ScrollSnap turns a scrolling session into a clean set of screenshot frames.

Start a capture, scroll naturally through a webpage or browser-rendered document, and ScrollSnap saves ordered PNG frames automatically to Chrome Downloads. Each session gets its own folder, with smart filenames, a `manifest.json` file, and a Markdown report.

Use ScrollSnap for product research, UX reviews, competitor teardowns, class notes, documentation, job research, and web research.

### Key Features

- Capture frames while you scroll
- Save PNGs automatically to Chrome Downloads
- Smart session and frame filenames
- Local session library
- Optional Markdown, ZIP, and PDF exports
- Local-only storage
- No backend
- No analytics
- No cloud sync

### Privacy

ScrollSnap captures only when you start a session. Captured screenshots are saved locally through Chrome Downloads. ScrollSnap does not upload, sync, sell, or share your screenshots.
